img tag

src 속성 : 이미지 파일의 경로

- alt 속성 : 이미지가 없을 때 표시되는 값
- 스크린 리더(화면 낭독기)가 이미지를 대신해서 읽을 갑을 작성할 용도로 사용함

width와 height 속성

    - 고정 크기 단위 : px, pt, cm
    - 가변 크기 단위 : %, em, rem

고정 크기 단위 : 화면의 크기와 관련 없이 요소의 크기를 항상 일정하게 유지

가변 크기 단위 : 화면의 크기와 또는 감싸고 있는 요소의 크기에 따라 변경됨 (주로 각 기기에따라 다르게 나오게할때 사용 view port)

이미지 구역을 2개로 나누어 각각 링크 설정
